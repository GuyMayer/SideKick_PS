## Why this matters

SideKick PS works out of the box, but every studio is different. The Settings
panel lets you change how the toolbar looks, which buttons appear, what your
QR codes show, how invoices sync, and where files go. You do not need to
touch most of these tabs — but knowing what is there means you can tweak
things as your workflow evolves.

## Opening Settings

Click the **⚙ Settings** gear icon on the toolbar, or press **Ctrl+Shift+I**
from anywhere.

The Settings window has a sidebar on the left with tabs. Click any tab to see
its options on the right.

## What each tab controls

| Tab | What you change here | When you would visit |
|-----|----------------------|---------------------|
| **⚙ General** | Start with Windows, tray icon, dark mode, auto-update, send activity logs | First setup — decide how SideKick PS behaves day-to-day |
| **🔗 GHL Integration** | GoHighLevel API key, Location ID, auto-tagging, email/SMS templates, invoice settings | When you change your GHL account or want to add automatic tags to contacts |
| **⌨ Hotkeys** | Keyboard shortcuts: Get Client (Ctrl+Shift+G), PayPlan (Ctrl+Shift+P), Settings (Ctrl+Shift+I) | When the default shortcuts clash with another programme you use |
| **📁 File Management** | SD card drive, download path, shoot archive location, file browser choice, room capture folder | Before your first SD card download — tell SideKick PS where your files live |
| **🎛 Toolbar** | Which buttons appear, icon colour, toolbar scale, auto-background detection, quick print printer, PDF output | When you want to hide buttons you never use, change icon colours, or set up PDF printing |
| **🖨 Print** | Print template selection (PayPlan vs Standard), print button calibration | When you want to change which template Quick Print uses, or calibrate the PDF click position |
| **🖥 Display** | QR code slides (Wi-Fi, mandate links, review pages), bank transfer details, custom images, display size and monitor | Before your first client session — set up what clients see on the big screen |
| **💳 GoCardless** | Enable/disable, API token, email and SMS templates for mandate requests | Before using Direct Debit for the first time |
| **📮 Cardly** | API key, card design, message template, test mode | Before sending your first postcard |
| **🔊 Sounds** | Enable/disable interface sounds, pace button sounds | When you want quieter operation or custom sound effects |
| **🔑 License** | Enter or update your licence key, see activation status | When you purchase or renew |
| **ℹ About** | Version number, build date, check for updates, What's New, diagnostics, **AI Help link** | When checking for updates or troubleshooting with support |

## A few settings worth changing early

- **Toolbar → Icon Colour** — switch to White, Black, Yellow, or any custom
  colour. The toolbar updates live as you choose.
- **Toolbar → Toolbar Scale** — make the toolbar smaller or larger to suit
  your screen.
- **Display → Slides** — set up your Wi-Fi QR and GoCardless mandate link
  before your first sale. The display is one of the most impressive things
  clients see.
- **File Management → Shoot Archive Path** — tell SideKick PS where your
  finished shoots live so it can find client images for postcards.

## What success looks like

Settings are saved automatically. Close the window and your changes take
effect immediately. The toolbar stays how you left it, your QR codes are
ready, and everything works the way your studio works.
