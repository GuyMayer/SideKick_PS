## Why this matters

Your client should never have to squint at your laptop screen to scan a
Wi-Fi code or read bank details. SideKick PS puts the information on your
big presentation monitor — full screen, easy to scan from across the room.
It makes your studio feel professional and keeps the focus on the client.

## What you can display

The full-screen display can show up to three slides. You choose what goes on
each one in **Settings → Display**:

- **QR codes** — enter any URL or text (Wi-Fi credentials, GoCardless mandate
  links, your website, a Google review page).
- **Bank transfer details** — your bank name, account holder, sort code, and
  account number displayed large and clear.
- **Custom images** — your logo, a welcome slide, seasonal offers — any image
  file.

Each slide is optional. If a slide is empty, it is skipped when you cycle
through.

## Using the display

1. Click the **QR Code** button (grid icon, teal) on the toolbar.
2. The display appears full-screen on your presentation monitor.
3. Use the **↑ / ↓ arrow keys** to move between slides.
4. Use the **← / → arrow keys** to move the display to a different monitor
   if you have more than one screen.
5. Press **Escape** or click the **QR Code** button again to close it.

## Setting up your slides

Go to **Settings → Display**. You will see three slide fields.

### For a QR code slide
Enter a URL. SideKick PS generates the QR code automatically. Common uses:
- `WIFI:T:WPA;S:StudioWiFi;P:yourpassword;;` — a Wi-Fi login code
- Your GoCardless mandate invitation link
- A link to your Google review page

### For bank transfer details
Fill in the fields at the bottom of the Display tab: bank name, account
holder, sort code, and account number. When any of these fields have content,
a "Bank Transfer" slide appears in the rotation automatically.

### For a custom image
Choose an image file (JPG or PNG). SideKick PS displays it full-screen as
one of the slides.

## Sizing and positioning

- **Display Size** — controls how large the QR code or image appears on
  screen (as a percentage of the screen size). Increase it if clients have
  trouble scanning.
- **Bank Scale** — controls the font size of bank details independently.
- **Monitor** — sets which screen the display appears on by default. You can
  still move it with the arrow keys during use.

## What success looks like

Your presentation monitor shows a clean, full-screen QR code or bank detail
slide. The client scans it from their phone without leaning in. You cycle
between slides smoothly with the arrow keys.

## When things go wrong

- **QR code is hard to scan** — Increase the **Display Size** in Settings.
  Also check the lighting — very bright rooms can make screens harder for
  phone cameras to read.
- **Slide appears blank** — No content has been set for that slide. Go to
  **Settings → Display** and add a URL, text, or image.
- **Display is on the wrong monitor** — Use the **← / →** arrow keys while
  the display is open to move it. You can also set the default monitor in
  **Settings → Display → Monitor**.
